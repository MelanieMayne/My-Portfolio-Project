My Projects

Astrology Season Weather App

HTML / CSS / React JS

I built and designed this astrology season weather app using HTML, CSS, React JS, Bootstrap, (Astrology Icons), (npm downloads for weather icons and sun sign), Weather API, VS Code, GitHub, and Netlify. The application can search for anywhere in the world and accurately provide the current weather as well as a five day forecast. It also tells you what Astrological Season we are in currently according to Western Astrology.

This was the final project in the SheCodes React Workshop.

This project is best viewed on a standard PC/Desktop Screen.

World Clock App

HTML / CSS / Vanilla JS

I built and designed this world clock application with HTML, CSS, Vanilla JS, Moment, VS Code, GitHub, and Netlify. The application provides the accurate date and time for nine cities all around the world as well as for your current location. It also automatically updates the time every second. By clicking on the World Clock heading on the app, it will take you back to the home page.

This was the final project in the SheCodes Plus Add-On Workshop.

This project can be viewed on all devices.
